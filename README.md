## How to Fork This Project

1. **On GitHub:**
   - Visit the repository page (create a new repo and upload this project if not already on GitHub).
   - Click the **Fork** button at the top right.
   - This will create a copy of the project in your own GitHub account.
2. **Clone Your Fork:**
   - Copy the URL from your forked repo (e.g., `https://github.com/yourusername/sky-journal.git`).
   - Open a terminal and run:
     ```sh
     git clone https://github.com/yourusername/sky-journal.git
     ```
3. **Install Requirements:**
   - Navigate to the project folder and run:
     ```sh
     python Sky\ journal.py
     ```
   - All dependencies will auto-install on first run.
